composer 安装和使用，附带替换国内镜像源

嗝嗝 7/8/2019 composer

Packagist.org 中国区全量镜像站点：https://php.cnpkg.org/ 使用参考文档：https://mirrors.aliyun.com/composer/index.html

# 安装

# linux/nuix 安装

# 下载

wget https://dl.laravel-china.org/composer.phar -O /usr/local/bin/composer
1

# 设置权限

chmod a+x /usr/local/bin/composer
1

如遇权限不足，可使用 sudo 权限执行

# windows 安装

  1. 直接下载 composer.phar

    地址：https://dl.laravel-china.org/composer.phar

  2. 把下载的 composer.phar 放到 PHP 安装目录

  3. 新建 composer.bat, 添加如下内容，并保存：

    @php "%~dp0composer.phar" %*
    1

  4. 鼠标右键文件，执行

# 设置国内源

# 设置PHP 包仓库 Packagist.org 中国区全量镜像站点.

  • Composer 全局安装配置

    composer config -g repos.packagist composer https://php.cnpkg.org
    1

  • 项目 composer.json 安装配置

    composer config repos.packagist composer https://php.cnpkg.org
    1

# 阿里云镜像源

  • 参考文案：点我

  • 全局配置

composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/
1
  • 项目配置
composer config repo.packagist composer https://mirrors.aliyun.com/composer/
1

# 取消配置

  • 取消全局配置
composer config -g --unset repos.packagist
1
  • 取消配置
composer config --unset repos.packagist
1
