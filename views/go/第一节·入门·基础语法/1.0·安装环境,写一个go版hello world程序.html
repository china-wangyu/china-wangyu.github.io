# 第一个hello world

go语言的入口文件必须以，package main为包名，不然go test或go build执行不了

  1. 打开编辑器，新建一个go项目

  2. 新建一个hello.go的文件

  3. 编辑hello.go，内容为：

    package main

import "fmt"

func main() {
    fmt.Printf("Hello World~")
}
